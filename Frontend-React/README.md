# 🛋️ FurniStyle: Professional E-Commerce Furniture Store

**FurniStyle** is a high-performance, responsive e-commerce platform built for a premium furniture shopping experience. It features a minimalist design, intuitive navigation, and a robust product management system designed to convert visitors into customers.

---

## 🚀 Project Overview

The goal of this project was to create a digital showroom that mirrors the elegance of high-end furniture. It provides users with a seamless flow from product discovery to checkout, emphasizing high-quality imagery and a clean, distraction-free User Interface (UI).

## ✨ Key Features

* **Dynamic Product Catalog:** Categorized browsing for Living Room, Bedroom, Office, and Kitchen furniture.
* **Advanced Filtering:** Sort by price, material, and popularity to find the perfect piece.
* **Interactive Shopping Cart:** Real-time quantity updates and persistent storage using Local Storage and Context API.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views using Bootstrap 5.
* **User Authentication:** Secure Sign-In and Sign-Up flows with comprehensive form validation.
* **Product Details:** In-depth views featuring high-resolution image galleries and technical specifications.

## 🛠️ Tech Stack

* **Frontend:** React.js (Functional Components & Hooks)
* **Styling:** Bootstrap 5 & Custom CSS3
* **Routing:** React Router v6
* **Icons:** Bootstrap Icons
* **State Management:** React Context API
* **Deployment:** Vercel / Netlify

## 📂 Project Structure

```text
src/
├── assets/             # Brand logos and static images
├── components/         # Reusable UI components (Navbar, Footer, ProductCard)
├── context/            # Global state management (Cart & Auth)
├── pages/              # Main views (Home, Shop, Account, Checkout)
├── data/               # Mock JSON data for products
└── styles/             # Global CSS and Bootstrap overrides
```

## ⚙️ Installation & Setup

Follow these steps to get the project running locally:

1.  **Clone the repository:**
    `git clone https://github.com/your-username/furniture-ecommerce.git`

2.  **Navigate to the project directory:**
    `cd furniture-ecommerce`

3.  **Install dependencies:**
    `npm install`

4.  **Start the development server:**
    `npm start`

5.  **Open the browser:**
    Visit `http://localhost:3000` to view the application.
